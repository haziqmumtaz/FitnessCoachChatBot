## Quick Start

1. **Install dependencies:**

   ```bash
   pnpm install
   ```

2. **Configure environment:**

   ```bash
   cp .env.example .env
   ```

   Then edit `.env` to set `VITE_API_BASE_URL` to your backend URL (e.g., `http://localhost:3001`)

3. **Start development server:**

   ```bash
   pnpm dev
   ```

4. **Open the app:**
   Visit `http://localhost:5173` in your browser
